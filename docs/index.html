Gin

Windows Installer creator

ginrequired

The root element of a gin bundle

Must contain a single dependency (system, meson, autotools, script)

id bundle id in reverse-DNS required
Valid in: N/A
namerequired

The name displayed to the user in the installer and add/remove programs

Valid in: gin
manufacturerrequired

The manufacturer name for add/remove programs

Valid in: gin
versionrequired

The version of the application bundled

Valid in: gin
dependencies

A list of dependencies which must be built before the current dependency

Valid in: system meson autotools script
system(dependency)

A package from the msys2 repos

name the package name without the arch prefix required
build-only marks the dependency as only being required whilst building the bundle
Valid in: gin dependencies
meson(dependency)

A dependency built with meson

See flag for passing flags to meson

name id used internally required
build-only marks the dependency as only being required whilst building the bundle
Valid in: gin dependencies
autotools(dependency)

A dependency built with autotools

See flag for passing flags to configure

name id used internally required
build-only marks the dependency as only being required whilst building the bundle
Valid in: gin dependencies
script(dependency)

A dependency built with a script

name id used internally required
path the (bash) script to use required
build-only marks the dependency as only being required whilst building the bundle
Valid in: gin dependencies
flag

A flag to pass to the build system (meson, autotools)

name the flag name required
Valid in: meson autotools
cleanups

A list of cleanups to apply after the build

Valid in: system meson autotools script
dir

A directory to remove from the bundle

path the directory to remove required
Valid in: cleanups
file

A file to remove from the bundle

path the file to remove required
Valid in: cleanups
exports

A list of things to export to the system

Valid in: system meson autotools script
shortcut

A shortcut to install in the start menu

Shortcut name taken from name

Contains a path to a executable

icon an icon to use instead of that embedded in the target
Valid in: exports
git(source)

A git repo to checkout

url where the repo is required
commit a specific commit to use
tag the tag to use
branch the branch to use
Valid in: meson autotools script
archive(source)

An archive to fetch

url where the archive is required
sha-256 a hash to validate the archive
Valid in: meson autotools script
patch

A patch file to apply to a source

path the patch file to apply required
Valid in: git archive
